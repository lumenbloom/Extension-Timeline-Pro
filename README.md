# Timeline Pro

> 为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 打造的交互式时间线图扩展——将你的聊天历史渲染成可导航的分支节点图。

---

## 📦 安装

```
SillyTavern → Extensions → Install Extension
https://github.com/lumenbloom/Extension-Timeline-Pro
```

或手动克隆到 `SillyTavern/data/default-user/extensions/`。

---

## 🚀 快速上手

- 打开任意对话 → 扩展面板 **Timeline Pro** → **Show Timeline**
- 或在聊天框输入 `/tl`
- 点击 **×** 或模态框外部关闭

---

## ✨ 亮点功能

### 交互式时间线图谱

- **交互式图例** — 图谱右下角展示所有角色和书签颜色，悬停高亮预览，点击锁定筛选
- **金色路径** — 打开时间线时，当前会话路径自动高亮标记，一眼定位

### 多会话 / 分支导航

- 在图中点击任意节点，可跳转到对应消息的会话位置
- **群聊支持** — 群聊中所有成员的对话历史同样可浏览

### 节点交互

- **单击** — 弹出详情面板：发送者、时间戳、消息内容、滑动轮次、全部分支列表；点击即可跳转或创建分支
- **双击** — 快速跳转到该消息
- **悬停预览** — 鼠标悬停节点 250ms 后显示消息摘要

### 书签路径高亮

- 书签标记的消息到根节点的整条路径以书签颜色高亮，边线加粗，层级提升，方便追溯关键节点

### 斜杠命令

- `/tl` — 打开时间线
- `/tl r` — 强制重载图谱

---

## ⚙️ 设置

| 设置项 | 说明 | 默认 |
|--------|------|------|
| Lock Nodes | 禁止拖动节点 | ✅ |
| Fixed Tooltip | 固定工具提示位置 | ❌ |
| Show Legend | 显示交互式图例 | ✅ |
| Bookmark Color | 书签路径颜色 | #ff0000 |

更多颜色、形状、间距等设置在抽屉面板中调整。

# 📖 English

> An interactive timeline graph extension for [SillyTavern](https://github.com/SillyTavern/SillyTavern) — visualize your chat history as a branching, navigable node graph.

---

## 📦 Installation

```
SillyTavern → Extensions → Install Extension
https://github.com/lumenbloom/Extension-Timeline-Pro
```

Or clone manually into `SillyTavern/data/default-user/extensions/`.

---

## 🚀 Quick Start

- Open any chat → Extension panel **Timeline Pro** → **Show Timeline**
- Or type `/tl` in the chat input
- Click **×** or outside the modal to close

---

## ✨ Highlights

### Interactive Timeline Graph

- **Interactive Legend** — Shows all characters and bookmark colors in the bottom-right corner. Hover to preview, click to lock filtering
- **Gold Path** — The current conversation's path is auto-highlighted on open, so you always know where you are

### Multi-Session / Branch Navigation

- Click any node to jump to the corresponding message in its chat session
- **Group Chat Support** — Browse all members' conversations in group chats

### Node Interactions

- **Click** — Opens a detail panel with sender, timestamp, message content, swipe count, and full branch list. Navigate to any session or create a new branch
- **Double-click** — Quick jump to the message
- **Hover Preview** — Message summary appears after 250ms

### Bookmark Path Highlighting

- The full path from a bookmarked message to the root is highlighted in the bookmark's color, with thicker edges and higher z-index — trace key story beats at a glance

### Slash Commands

- `/tl` — Open the timeline
- `/tl r` — Force-reload the graph

---

## ⚙️ Settings

| Setting | Description | Default |
|---------|-------------|---------|
| Lock Nodes | Prevent node movement | ✅ |
| Fixed Tooltip | Pin tooltip position | ❌ |
| Show Legend | Display interactive legend | ✅ |
| Bookmark Color | Bookmark path color | #ff0000 |

More color, shape, and spacing settings available in the drawer panel.
